# Rohindia Static Website

Static HTML/CSS/JS version of the Rohindia website.

## Pages
- `index.html` - Home page
- `vision.html` - Vision page
- `founders_desk.html` - Founder's message

## Running locally
Simply open `index.html` in a browser, or use a simple server:
```bash
npx serve .
```

## Development
Edit the HTML files directly. CSS/JS are in `css/` and `js/` directories.

## Notes
- Uses FontAwesome CDN for icons
- Pure static - no backend, no build step required
- API calls will be added later
